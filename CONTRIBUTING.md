# 🤝 راهنمای مشارکت

از اینکه می‌خواهید در NexusChat مشارکت کنید سپاسگزاریم! ✨

## 🐛 گزارش Bug

اگه باگی پیدا کردید:
1. ابتدا [Issues](https://github.com/mahan80api/NexusChat/issues) رو جستجو کنید
2. اگه وجود نداشت، یه Issue جدید باز کنید
3. عنوان واضح، توضیحات کامل، و مراحل reproduction بنویسید

## 💡 پیشنهاد Feature

1. یه Issue با برچسب `enhancement` باز کنید
2. توضیح بدید چرا این feature لازمه
3. اگه می‌تونید، mockup یا example اضافه کنید

## 🔧 Pull Request

### مراحل
1. Fork کنید
2. Branch جدید بسازید: `git checkout -b feature/my-feature`
3. تغییرات رو commit کنید: `git commit -m "feat: add my feature"`
4. Push کنید: `git push origin feature/my-feature`
5. Pull Request باز کنید

### قوانین کدنویسی
- **PHP:** PSR-12 standard
- **JavaScript:** ES6+، بدون jQuery
- **CSS:** BEM-like naming
- **SQL:** lowercase keywords
- **Commit:** Conventional Commits

### قبل از PR
- [ ] کد تست شده
- [ ] مستندات به‌روز شده
- [ ] بدون conflict
- [ ] CHANGELOG.md به‌روز شده

## 📝 Commit Convention

```
feat: add new feature
fix: fix bug
docs: update docs
style: formatting only
refactor: code restructuring
test: add tests
chore: build/config update
```

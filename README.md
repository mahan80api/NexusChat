# 🌌 NexusChat

A premium, professional messaging platform built with **PHP + MySQL** featuring a stunning **8D Galaxy Theme**.

## ✨ Features

- 💬 Real-time messaging (private, group, channel)
- 🔐 End-to-End Encryption (RSA-2048)
- 📞 Voice & Video Calls (WebRTC)
- 📸 Stories / Status (24h expiry)
- 🎨 8D Galaxy Theme with particle effects, glassmorphism & 3D parallax
- 📁 File & Media sharing
- 👥 Group chats & Channels
- 🔔 Smart notifications
- ⚡ Typing indicators & online status
- 📌 Pinned messages & replies
- 😊 Reactions & emoji
- 🌓 Dark / Light / Galaxy themes
- 📱 Fully responsive

## 🛠 Tech Stack

- Backend: PHP 8+ (OOP, MVC)
- Database: MySQL 5.7+ / MariaDB
- Frontend: Vanilla JS, CSS3
- Real-time: AJAX Long Polling
- Calls: WebRTC
- Crypto: OpenSSL (RSA-2048)

## 🚀 Installation

1. Clone the repository
2. Move to XAMPP/Laragon htdocs folder
3. Import database: `mysql -u root -p < database.sql`
4. Configure `config/database.php` with your DB credentials
5. Open: `http://localhost/NexusChat`
6. Register and enjoy! ✨
